# mush-go

ospitare dentro la cartella packages tutti una serie di pachetti scritti in go che posi saranno installabili con mush ad esempio

usare per ogni file il `build.sh` per eseguire l'installazione in se del packetto, capire se fare lo shim con il `src/main.sh` oppure no

```
mush install vale.sh
```

Info: <https://vale.sh/>
